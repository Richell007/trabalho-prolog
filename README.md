# trabalho-prolog
Sistema especialista em Prolog com adaptação dinâmica de dificuldade, análise de erros por tema e feedback personalizado de aprendizado.

Um sistema de quiz inteligente desenvolvido em Prolog que se adapta dinamicamente ao nível de conhecimento do jogador, oferecendo uma experiência personalizada de aprendizado.

## ✨ Características Principais

- 🧠 **Sistema Adaptativo**: A dificuldade das perguntas aumenta conforme o jogador acerta
- 📊 **Análise de Desempenho**: Identifica temas com mais dificuldade e foca neles
- 🎯 **Foco Inteligente**: Prioriza perguntas dos temas onde o jogador tem mais erros
- 📈 **Feedback Detalhado**: Ao final, gera análise completa de desempenho
- 🎮 **50 Perguntas**: Banco diversificado dividido em 3 níveis e 6 temas

## 🗂️ Temas Abordados

| Tema | Descrição |
|------|-----------|
| 🌍 Geografia | Capitais, localizações, características geográficas |
| 🧮 Matemática | Operações básicas, raiz quadrada, cálculo diferencial/integral |
| 🔬 Ciência | Física, química, biologia, astronomia |
| 📜 História | Eventos históricos, datas importantes |
| 📚 Literatura | Obras, autores clássicos e modernos |
| 🎲 Geral | Conhecimentos diversos e curiosidades |
